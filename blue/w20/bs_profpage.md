## Tips/Links

- A basic diagram of the above page is like

```
I: jumbotron

	> sub text

II: body container
> column
	> bla text
> column
	> bla text
> column
	> bla text
```

- Remember, 12 units per row.  You can stagger class sizes, like...

```html
<div class="container">
  <div class="row">
    <div class="col">
      A column
    </div>
    <div class="col-5">
      A column
    </div>
    <div class="col-4">
      A column
    </div>
  </div>
</div>
```
